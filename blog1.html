What Programming Language is Used in Arduino?

Arduino uses a programming language based on C++, which is a simplified version of C++ tailored to make it beginner-friendly. It is implemented through the Arduino IDE (Integrated Development Environment) and relies on libraries and functions specific to Arduino hardware, like digitalWrite() and analogRead().

Things I Know About Robotics

Things I Know:

  • Robotics involves building machines that can automate tasks or respond to commands, often by combining hardware (sensors, motors, controllers) and software.
  • Programming is the foundation of instructing these machines using languages like Python, C++, or Java.
  • Microcontrollers like Arduino are essential in robotics for controlling electronic components like LEDs, motors, and sensors.

Things I Want to Learn:

  • How to build a functional robot, starting from basic circuits to advanced designs.
  • Mastering Arduino programming, especially creating autonomous systems using sensors and motors.
  • Diving into Artificial Intelligence (AI) for robotics, enabling robots to learn and adapt to their environments.
